What is an f-string?

An f-string, or formatted string literal, is a way to embed expressions inside string literals in Python.

Basic Syntax:

Example: f"My name is {name}, and I am {age} years old."

Place variables or expressions inside curly braces {} within a string.

Variable Inclusion:

Embed variable values directly in the string.

Example: f"My age is {age}."

Expression Inclusion:

Include expressions or calculations inside curly braces.

Example: f"The result is {5 + 3}."

Formatting Numbers:

Control the display of numbers with formatting options.

Example: f"Value of pi: {pi:.2f}" (Formats pi to 2 decimal places).

Escape Characters:

Use double curly braces {{}} to include literal curly braces in the string.

Example: f"{{Hello}}" produces "{Hello}" in the output.

Concatenation:

Easily concatenate strings and variables within an f-string: f"Hello, {name}!".

Older Python Versions:

F-strings are available from Python 3.6 onwards.

For compatibility with older versions, consider using % formatting or str.format().
