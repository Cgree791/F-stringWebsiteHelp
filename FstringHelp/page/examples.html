Examples

Expressions and Functions


// Calculate and display the area of a circle using an f-string
radius = 5
area = f"The area of a circle with radius {radius} is {3.14 * radius ** 2}."
print(area)
Calculating and displaying the area of a circle based on the provided radius.

Boolean Values


// Check if Python is fun and print the result using an f-string
is_python_fun = True
feedback = f"Is Python fun? {is_python_fun}"
print(feedback)
Checking and displaying whether Python is considered fun.

Date and Time


// Get the current date and time and format it using an f-string
from datetime import datetime

current_date = datetime.now()
formatted_date = f"Current date and time: {current_date:%Y-%m-%d %H:%M:%S}"
print(formatted_date)
Fetching and displaying the current date and time in a specific format.

Multiline f-strings


// Displaying information in a multiline format using f-strings
name = "John"
age = 30
address = "123 Main Street"

info = f"""
Name: {name}
Age: {age}
Address: {address}
"""
print(info)
Using f-strings to create a multiline output with personal information.

Using F-strings with Dictionaries


// Using f-strings to format information stored in a dictionary
person_info = {"name": "Alice", "age": 25, "city": "Wonderland"}
details = f"Person: {person_info['name']}, Age: {person_info['age']}, City: {person_info['city']}"
print(details)
Formatting dictionary values using f-strings.

Combining with Other String Methods


// Combining f-strings with other string methods
message = "hello"
formatted_message = f"{message.capitalize()} world!"
print(formatted_message)
Using f-strings with the capitalize method to modify a message.

Error Handling


// Handling errors and displaying a custom error message using an f-string
value = "abc"
try:
    number = int(value)
except ValueError as e:
    error_message = f"Error: {e}. Couldn't convert '{value}' to an integer."
    print(error_message)
Handling a ValueError and providing a custom error message using f-strings.

Unicode and Escape Sequences


// Displaying Unicode characters and escape sequences using f-strings
unicode_example = f"Unicode: \u03A9"
escape_sequence = f"New line: First line.\nSecond line."
print(unicode_example)
print(escape_sequence)
Using f-strings to display Unicode characters and escape sequences.

Nesting F-strings


// Nesting f-strings to create a more complex string
name = "Bob"
age = 28
nested_example = f"Nested example: {f'Name: {name}, Age: {age}'}"
print(nested_example)
Creating a nested f-string for a more complex output.
